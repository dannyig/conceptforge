1. Architecturally Closest Projects

shrimbly/node-banana Closest Twin
github.com/shrimbly/node-banana
Free, open-source, node-based generative AI workflow editor built with Next.js + TypeScript + React Flow. BYOK API key model. Has a CLAUDE.md — built with Claude Code assistance. MIT licence.
Study its node type system, handle typing, canvas interaction patterns, and CLAUDE.md structure. Closest architectural twin to ConceptForge.
FlowiseAI/Flowise 35k+ stars
github.com/FlowiseAI/Flowise
Most prominent open-source AI workflow builder using a React Flow drag-and-drop canvas. React + TypeScript + Node backend. Gold standard for node component architecture and edge type patterns.
Study the UI source for node component structure even if you don't need the backend.
langgenius/dify Reference
github.com/langgenius/dify
Production-grade agentic workflow builder with a React Flow visual canvas. Large-scale open-source project. Good reference for how a mature team structures complex canvas-based AI apps with React + TypeScript.
claudio-silva/claude-artifact-runner Stack Match
github.com/claudio-silva/claude-artifact-runner
Vite + React 18 + TypeScript + Tailwind template mirroring Claude's Artifact execution environment. No backend. Deploys to Vercel or Cloudflare Pages. Bootstrap with npx run-claude-artifact.
Exact stack match — use as a structural reference for ConceptForge's Vite + React + no-backend setup.
xyflow/xyflow (React Flow) Core Library
github.com/xyflow/xyflow · reactflow.dev/showcase
The React Flow library itself. The Showcase page lists real-world apps built with React Flow. Pro Examples include workflow builders and canvas apps directly applicable to ConceptForge.

2. Claude Code Workflow & Dev Method

Pimzino/claude-code-spec-workflow Slash Commands
github.com/Pimzino/claude-code-spec-workflow
Automated slash-command workflows for Claude Code: Requirements → Design → Tasks → Implementation and Bug Fix → Analyze → Fix → Verify cycles.
Directly applicable to ConceptForge's requirement ID-driven devmethod. Integrate these slash commands into the project workflow.
github/spec-kit GitHub Official
github.com/github/spec-kit
GitHub's official Spec-Driven Development toolkit for Claude Code. Produces structured spec artifacts (requirements, design, tasks) that guide agents. Integrates with CLAUDE.md workflows.
ChrisWiles/claude-code-showcase Reference
github.com/ChrisWiles/claude-code-showcase
Comprehensive Claude Code project configuration example covering hooks, skills, custom agents, slash commands, and GitHub Actions. Stack includes React + TypeScript. Good CLAUDE.md structure reference.
senaiverse/claude-code-reactnative-expo-agent-system Multi-Agent
github.com/senaiverse/claude-code-reactnative-expo-agent-system
7 production sub-agents for React apps: accessibility, design systems, security, performance, testing. Built for Claude Code v2.0.5+. Reference for defining specialised agent roles.

3. Official Anthropic Resources

frontend-design Claude Code Plugin Install First
github.com/anthropics/claude-code/tree/main/plugins/frontend-design
Official Claude Code skill that guides agents toward distinctive typography, bold colour palettes, and high-impact UI design. Directly prevents generic "AI slop" aesthetics. Essential for ConceptForge's dark theme.
Install before starting UI development: 
claude code skills install frontend-design
Improving Frontend Design Through Skills — Anthropic Blog Official
claude.com/blog/improving-frontend-design-through-skills
Official blog post introducing the frontend-design skill with dramatic before/after UI examples. Read this to understand what the skill changes and why it matters.
Anthropic Cookbook — Prompting for Frontend Aesthetics Official
platform.claude.com/cookbook/coding-prompting-for-frontend-aesthetics
Official guide for prompting Claude to produce quality frontend UI output. Read before writing any UI-related agent prompts for ConceptForge.
Claude Code Skills Documentation Docs
code.claude.com/docs/en/skills
Official documentation for installing and creating Claude Code skills. Required reading for managing the frontend-design skill and any custom skills added to ConceptForge.
